# BMI Hesaplayıcı

SwiftUI ile geliştirilmiş basit bir Vücut Kitle İndeksi (BMI) hesaplayıcı.

## Özellikler

- Boy ve kilo girişi
- BMI hesaplama
- Kategori belirleme (Zayıf, Normal, Fazla Kilolu, Obezite)
- İdeal kilo aralığı gösterimi
- Renk kodlu sonuç gösterimi
- Klavye kontrolü

## Gereksinimler

- iOS 17+
- Xcode 15+
- Swift 5.9+

## Kurulum

1. Repo'yu klonla:
git clone https://github.com/safakyusuf/bmi-hesaplayici.git

2. Xcode ile projeyi aç
3. Simülatörde veya cihazda çalıştır

## Öğrenme Notu
